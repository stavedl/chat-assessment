# Chat Assessment · AI 对话内嵌心理量表测评

[English](./README.md) | 简体中文

在 AI 对话中以自然对话的方式完成标准化心理量表测评的演示项目：用户主动提出测评，或系统根据对话信号主动推荐，全流程在对话框内完成「触发 → 确认 → 逐题施测 → 计分 → 分层报告」。

**在线演示**：https://stavedl.github.io/chat-assessment/

## 功能特性

- **主动触发**：识别用户测评意图（"我想做个测评"），确认后施测
- **被动触发**：四类信号（关键词 / 语义 / 情绪 / 行为）累积评分，达阈值且通过四道决策闸门（危机旁路、时机、频控冷却、用户偏好）后以建议方式推荐，右侧面板实时可视化
- **危机旁路**：检测到自伤等危机表达时拦截测评推荐，切换危机干预话术
- **对话式施测**：逐题呈现 + 选项按钮，支持文字模糊作答、上一题修改、最小作答间隔、随时退出
- **计分与报告**：因子公式计分、得分区间分档，输出解读 + 专家建议 + 免责声明，全同分作答预警
- **配置驱动**：量表以标准 JSON 配置接入，新增量表无需改代码

## 演示量表

接纳与行动问卷第二版（AAQ-II，编码 hs1051），单因子「经验性回避」，7 题 7 点计分，总分 7–49：

| 得分 | 档位 |
|---|---|
| 7–12 | 较低 |
| 13–28 | 中等 |
| 29–49 | 较高 |

## 本地运行

无需安装任何环境，下载 `index.html` 后用浏览器打开即可。

## 使用方式

打开页面后，点右侧「场景一」体验主动触发，「场景二」自动播放一段倾诉对话并观察信号累积与推荐过程；也可直接在输入框聊天（含关键词的消息会累积触发信号）。
